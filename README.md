# LongLivedTriggers
Preliminary studies of triggers for Long Lived particles

## Git instructions

In your working area, first set up the CMSSW release and initialize git:
```bash
cmsrel CMSSW_9_2_10
cd CMSSW_9_2_10/src
cmsenv
git cms-init
```

Clone the repository:

```bash
cd $CMSSW_BASE/src
mkdir LongLived
cd LongLived
git clone https://github.com/lbenato/LongLivedTriggers
```
